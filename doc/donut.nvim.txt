1. Donut.nvim                                          *donut.nvim-donut.nvim*


  donut.c meets Neovim
Donut.nvim is the Neovim screensaver that spawns a spinning donut in each
opened Neovim window.

**Note:** donut.nvim does not change any of the buffers, it just creates new
buffers on top of existing ones and after you press any key it restores your
buffers.

INSTALLATION                              *donut.nvim-donut.nvim-installation*


  [!NOTE]
  - Neovim version >= 0.9.5 is required.
  - No need to call `setup` function, donut.nvim is ready to use out of the box.
  - Only call `setup` function if you want to change the default configuration.
  - No need to lazy load it via Lazy.nvim, donut.nvim lazy loads by default.
lazy <https://github.com/folke/lazy.nvim>

>lua
    {
        "NStefan002/donut.nvim",
        version = "*",
        lazy = false,
    }
<

rocks.nvim <https://github.com/nvim-neorocks/rocks.nvim>

`:Rocks install donut.nvim`


CONFIGURATION                            *donut.nvim-donut.nvim-configuration*

Default configuration ~

>lua
    {
        timeout = 300,
        sync_donuts = false,
    }
<

If you want to change the default configuration, you can call the `setup`
function.

>lua
    require("donut").setup({
        timeout = 60,
        sync_donuts = true,
    }
<

or you can just set `vim.g.donut_config` to a table with the configuration you
want.

>lua
    vim.g.donut_config = { timeout = 30, sync_donuts = false }
<
